About


This website contains biographical data for individuals interred in Oak Grove Cemetery (formerly Stonewall Jackson Memorial Cemetery) in Lexington, Virginia. The data was gathered by Thomas Kastner and the original project website was published by Carol Karsch (former Library Data and Statistical Support Specialist) with help from Emily Crawford '15 through Leyburn Library at Washington and Lee University. In 2023, the data was cleaned and republished on the current website by Mackenzie Brooks, Associate Professor & Digital Humanities Librarian.

A press release about project from 2013 is available.

Support for the original project was provided by:

  • Washington & Lee University Library
  • Rockbridge Historical Society
  • APVA/Preservation Virginia
  • City of Lexington

Data Documentation

The following modifications were made to the data set in 2023:
  • Birth and death dates were standardized in ISO I601 format, expressed as YYYY-MM-DD.
  • The fields Monument Type, Condition, and Veteran were standardized. The original data was inconsistent in the way it expressed terms like "Monument, double" or military service.
  • Images of gravestones are no longer available.
